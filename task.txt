file structures are like this;
geopolitical_portfolio/
│
├── main.py                        # Orchestration entry point
├── config.py                      # All constants: thresholds, tickers, params
├── requirements.txt
│
├── data/
│   ├── __init__.py
│   ├── loader.py                  # Phase 2 tech env: UTF-8-SIG, NFC normalization
│   └── universe.py                # KOSPI top-200 + S&P500/NASDAQ/DJI components
│
├── signals/
│   ├── __init__.py
│   ├── trend.py                   # Phase 1-A: MA 5/22/60/182 + Envelope ±10%
│   ├── macro.py                   # Phase 1-B: VIX kill-switch, TIPS sensitivity
│   └── composer.py                # Merges trend + macro → final Q and Ω inputs
│
├── optimization/
│   ├── __init__.py
│   ├── hrp.py                     # Phase 2-A: Riskfolio-Lib HRP clustering
│   ├── black_litterman.py         # Phase 2-B: BL views (Q) + Omega construction
│   └── cdar.py                    # Phase 2-C: CDaR constraint, UPI maximization
│
├── portfolio/
│   ├── __init__.py
│   ├── selector.py                # Top-10 asset selection + CSV/terminal output
│   ├── factor_loading.py          # Fama-French 5-Factor exposure per stock
│   └── constraints.py            # 20-day avg volume → liquidity weight caps
│
├── backtest/
│   ├── __init__.py
│   └── engine.py                  # vectorbt: 2008 / 2020 / 2026 scenario runs
│
└── evaluation/
    ├── __init__.py
    └── metrics.py                 # Serenity Ratio, Ulcer Index, Calmar Ratio

the implementation steps are like this;
config.py
    └── data/ (Step 2)
            └── signals/ (Step 3)
                    └── portfolio/factor_loading (Step 5a)
                            └── optimization/ (Step 4)
                                    └── portfolio/selector (Step 5b)
                                            └── backtest/ (Step 6)
                                                    └── evaluation/ (Step 7)
